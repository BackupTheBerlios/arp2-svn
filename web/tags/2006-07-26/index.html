ARP2 — Main page

Introduction

ARP2 is currently just a collection of open source software by Martin Blom. One day it may be something usable in itself ...

Sub-projects

  • AHI, the audio system for AmigaOS, AROS and MorphOS.
  • gg-cross, the Linux-hosted, Geekgadgets-based cross-compiler suite for m68k-amigaos, i686be-amithlon and ppc-morphos.
  • RDesktop for AmigaOS and MorphOS.
  • glgfx, an Amiga-like graphics system built on top of Linux, Xorg and OpenGL. (Pre-alpha quality, but do try the window_demo, it's pretty neat.)

Please visit the Developer Project Page for forums, source code, mailing lists, bug tracker etc.
